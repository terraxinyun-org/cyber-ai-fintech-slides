# Intro to Cybersecurity & AI for FinTech — Training Slides

Instructor slide deck for the **Intro to Cybersecurity & AI for FinTech (OpenCode Edition)** hands-on lab.

## Quick Start

Open `index.html` in any modern browser — no build step required.

```
# Or serve locally
npx serve .
```

## Navigation

| Key | Action |
|-----|--------|
| `→` / `Space` | Next slide |
| `←` | Previous slide |
| `S` | Open speaker notes (instructor view) |
| `F` | Fullscreen |
| `Esc` / `O` | Slide overview |
| `?` | Show keyboard shortcuts |

## Modules

| # | Module | Duration | Key Topics |
|---|--------|----------|------------|
| 01 | Threat Landscape | 20 min | CIA triad, common attacks, AI in security, SG regulations |
| 02 | Scan, Discover & Exploit | 45 min | nmap, nikto, SQL injection, XSS, IDOR, default credentials |
| 03 | Incident Response | 30 min | SIEM, Elasticsearch/Kibana, brute force investigation, IR report |
| 04 | Governance & Compliance | 25 min | MAS TRM, PDPA, stakeholder interview, gap analysis |
| 05 | Wrap-Up | 20 min | AI limitations, career paths, certifications, next steps |

**Total: ~2.5 hours**

## Speaker Notes

Press `S` to open the speaker notes window. Each slide includes instructor notes with:
- Talking points and teaching tips
- Questions to ask students
- Key messages to emphasize
